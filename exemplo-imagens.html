Como Usar no Dashboard

Para adicionar novas imagens ao dashboard, você pode:

  1. Colocar a imagem na pasta public/
  2. Referenciar usando o caminho public/nome-da-imagem.extensao
  3. Usar classes CSS para dimensionar e estilizar

Exemplo de Adição de Nova Imagem:

<!-- Adicionar no HTML -->
<img src="public/sua-imagem.jpg" alt="Descrição" class="h-8 w-8">

<!-- Adicionar no CSS -->
.h-8 { height: 2rem; }
.w-8 { width: 2rem; }
